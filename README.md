# Лабораторная работа №22: Интерактивный калькулятор на JavaScript

## 1. Описание проекта
Веб-приложение "Калькулятор", разработанное с использованием чистого HTML, CSS и JavaScript (ES6+). Проект демонстрирует принципы модульной архитектуры, объектно-ориентированного программирования в JS, работу с DOM и системой контроля версий Git/GitHub.

## 2. Функциональность
- **Поддерживаемые операции:** сложение (`+`), вычитание (`-`), умножение (`*`), деление (`/`), десятичные числа.
- **Ограничения:** запрещён ввод двух операторов подряд, нескольких точек в одном числе, выражений, начинающихся с оператора (кроме `-`), и выражений, заканчивающихся оператором.
- **Обработка ошибок:** деление на ноль, некорректный синтаксис и недопустимые символы перехватываются. При ошибке на дисплее выводится сообщение "Ошибка", а дальнейший ввод возможен только после очистки (`C`).

## 3. Архитектура
### Структура файлов
---index.html 
--- style.css 
--- calculator.js  
---main.js  

### Класс `Calculator`
Инкапсулирует всю логику вычислений и управления состоянием:
- `expression` (строка) — накопленное математическое выражение.
- `hasError` (boolean) — флаг состояния ошибки.
- Методы разделены по ответственности: `handleButtonClick()` (маршрутизация), `addToExpression()` (валидация), `calculate()` (вычисление), `safeEvaluate()` (безопасный вызов `eval`), `clear()`/`updateDisplay()`/`showError()` (управление UI).

### Взаимодействие компонентов
1. `index.html` загружает `main.js` как модуль (`type="module"`).
2. `main.js` импортирует класс `Calculator`, создаёт экземпляр, передавая DOM-элемент дисплея.
3. При загрузке `init()` навешивает обработчики событий на кнопки.
4. Клики передаются в `Calculator`, который обновляет состояние и вызывает `updateDisplay()`.

## 4. Логика работы
1. **Нажатие кнопки:** событие `click` передаётся в `handleButtonClick(value)`.
2. **Валидация ввода:** `addToExpression()` проверяет корректность символа относительно текущего состояния. Если проверка проходит, символ добавляется к `expression`, и дисплей обновляется.
3. **Вычисление:** при нажатии `=` вызывается `calculate()`. Метод проверяет валидность выражения, использует `safeEvaluate()` (с предварительной регулярной проверкой) для вычисления через `eval()`. Результат сохраняется в `expression`.
4. **Обработка ошибок:** при делении на ноль, синтаксических ошибках или невалидном вводе срабатывает `showError()`, устанавливается флаг `hasError = true`, дисплей показывает "Ошибка".

## Установка и запуск
1. Клонируйте репозиторий: `git clone <URL>`
2. Откройте папку проекта в VS Code.
3. Запустите через **Live Server** (ПКМ по `calculator/index.html` → `Open with Live Server`). 
⚠️ *Примечание: ES6-модули не работают при открытии файла напрямую через `file://` из-за политики CORS.*
